Overcoat

In-process successor to the Kometa-Jellyfin (jellymeta) script. Configure credentials and per-library behaviour, then run the "Apply Overcoat Overlays" scheduled task (Dashboard → Scheduled Tasks).

Credentials

TMDB v3 API key. Required for status, trending and Top 250 data.

Behaviour

Badges

Libraries

One row per library. Name must match the Jellyfin library exactly. Leave Type as "auto" to detect from Jellyfin. Disabled libraries are skipped.

Ignore titles

One title per line. Matching items are skipped entirely.

TMDB overrides

One per line as Title | year | tmdbId (year may be 0).
