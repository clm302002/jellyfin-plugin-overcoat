Schedule

When the overlay job runs automatically. Pick a quiet hour — on a large library the run can take a while.

On, Overcoat keeps the task on the daily time below. Off, it stops touching the schedule and you manage it yourself in Dashboard → Scheduled Tasks (intervals, several times a day, on startup). Turning this off does not stop the task running — any trigger already set there stays active. To stop automatic runs entirely, remove the triggers in Dashboard → Scheduled Tasks.

Status dates

AIRING shows the next episode in the chosen format, always (no "within" window). RETURNING shows the upcoming episode/season date only when it's within the chosen window. (Turn AIRING off in the Statuses group and an airing show falls back to RETURNING.)

The window can't be shown in the preview (it depends on real air dates) — only the format is previewed.

Colours

Tap a swatch to recolour that status. Switch the preview to that status to see it.

Statuses & labels

Untick to skip a status entirely; edit the text to rename it (e.g. RETURNING → RETURNS).

Live preview

Updates instantly — no run needed. Random picks a clean poster from your own library (click again for another); it never uses one Overcoat has already overlaid, so what you see is what a real run would produce.

Limit to specific shows/movies

One title per line. When non-empty, ONLY these titles are processed (great for testing). Leave empty to process everything in the enabled libraries.

Ignore titles

One title per line. Matching items are skipped entirely.
